# pos-system
小超市收银系统，包含前台和后台。

## 结构

服务端和客户端打包在一起，正常打包使用时软件打开客户端后会在后台静默启动服务端。

当局域网内有多台收银机时，可由其中一台充当服务端，其他收银机充当客户端。

在软件根目录下建立一个`.noserver`的空文件即可不启动服务端，打开客户端后在登录界面修改服务器地址即可。

打开`powershell`后执行如下命令即可:

```bash
echo "" > .noserver
```
